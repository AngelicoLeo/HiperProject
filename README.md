# HiperProject
Project developed with Blazor .NET 6 microservices, RabbitMQ and MongoDB,
Status: Ongoing 
<p align="center">
- This project has builded  with MongoDB for database, .net 6 and microservices idea, I used docker-compose to the web app and API, in the API I worked with the 
Repository's pattern and create a controller to comunicate with the RabbitMQ</p>
The current project are using the development settings, and the database are looking for a localhost, in the future maybe I'll upload to a free server (there is some free options for mongo)


### Features

- [x] CRUD
- [x] Database
- [x] RabbitMQ
